# h264-player

```sh
$ npm install
```

### Start the server

```sh
$ cd server
$ node ws_server.js
```


### Serve the client page
Create a HTTP server to serve the index page.

Here, python is being used to create the HTTP server.

Python 2:
```sh
$ python -m SimpleHTTPServer 8001
```

Python 3:
```sh
$ python3 -m http.server 8001
```

Browse: http://localhost:8001/
